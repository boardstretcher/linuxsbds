LinuxSBDS

Stop locking your development group into expensive big-box software of the month. Try using well-tested, free, mature, open-source software from around the internet, conveniently brought together in one place for you and your team. Supporting roles from the developer, through the tester, all the way up to the project manager -- linuxsbds is the way to do it.

Made of OpenSource Software:

  • git - In software development, Git /ɡɪt/ is a distributed version control and source code management (SCM) system with an emphasis on speed

  • gitlab - GitLab is a fast, secure and stable solution to manage your projects. It is based on Ruby on Rails and has a free and open-source license (MIT). GitLab is the most installed git management application in the world

  • mediawiki - mediawiki is a free software open source wiki package written in PHP, originally for use on Wikipedia

  • jenkins - Jenkins is an open-source continuous integration server with 300+ plugins to support all kinds of software development

  • testlink - TestLink is a web-based test management software that facilitates software quality assurance

  • watir - Watir, pronounced water, is an open-source (BSD) family of Ruby libraries for automating web browsers. It allows you to write tests that are easy to read and maintain. It is simple and flexible

  • zerobin - ZeroBin is a minimalist, opensource online pastebin/discussion board where the server has zero knowledge of hosted data. Data is encrypted/decrypted in the browser using 256 bits AES

  • gerrit - Gerrit is a web based code review system, facilitating online code reviews for projects using the Git version control system

  • mantis bugtracker - MantisBT is a free popular web-based bugtracking system (feature list). It is written in the PHP scripting language and works with MySQL, MS SQL, and PostgreSQL databases and a webserver

  • redmine - A flexible project management web application which includes a gantt chart, calendar, wiki, forums, multiple roles, and email notification

  • sqlbuddy - Web based MySQL administration

  • extplorer - eXtplorer is a web-based File Manager
